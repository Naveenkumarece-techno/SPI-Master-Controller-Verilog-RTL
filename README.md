# SPI Master Controller using Verilog HDL

## Overview
This project implements a parameterized SPI (Serial Peripheral Interface) Master Controller using Verilog HDL for serial communication between master and slave devices in digital systems and SoC architectures.

The design includes FSM-based control logic, SPI clock generation, MOSI/MISO data transfer mechanisms, and configurable communication architecture for synchronous serial data exchange.

The SPI Master Controller was verified using RTL simulation and waveform analysis in Vivado/ModelSim to ensure proper clock synchronization and serial communication functionality.

This project helped in understanding:
- SPI communication protocol
- FSM-based digital design
- Serial data transmission
- Clock division techniques
- RTL design methodology
- Testbench verification and simulation

## Features
- SPI Master Controller Design
- FSM-Based Control Logic
- MOSI/MISO Data Transfer
- SPI Clock Generation
- Parameterized Architecture
- RTL Design using Verilog HDL
- Functional Verification using Testbench
- Waveform Analysis in Vivado/ModelSim

## Tools Used
- Verilog HDL
- Xilinx Vivado
- ModelSim
